기준일: 2026-03-23

## 공통 원칙

- 기본은 **원서(EN)를 source of truth** 로 사용한다.
- 예외적으로 KO-only 책은 **KO 원본 기준**으로 웹 단위를 설계한다.
- 웹 요약 1개 목표 분량은 대략 **8k ~ 25k chars**.
- 4k chars 이하 단위는 앞뒤와 합치고, 30k chars 이상 단위는 내부 소제목 기준 분할한다.
- 원본 TOC는 참고만 하고, 실제 산출물은 **웹용 요약 단위(web chapter map)** 로 재구성할 수 있다.
- 책 하나를 시작할 때마다 이 초안을 그대로 따르지 않고, **착수 전 점검(메타데이터 / TOC / 실제 구조 / 분량 / 예외사항 확인)** 후 최종 작업 단위를 확정한다.
