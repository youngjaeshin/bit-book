# Korean glossary for Austrian economics / Bitcoin translation

- hard money → 경화 / 하드 머니
- fiat money / fiat currency → 법정화폐
- money printing / printing money → 화폐 발행 확대 / 돈 찍기
- monetary debasement → 화폐 가치 훼손 / 통화 희석
- monetary inflation → 통화 인플레이션
- price inflation → 물가 인플레이션
- purchasing power → 구매력
- store of value → 가치 저장 수단
- medium of exchange → 교환 매개
- unit of account → 회계 단위

## Preferred style notes

- sound money는 문맥상 거의 항상 **건전 화폐**를 우선한다.
- fiat는 맥락상 비판적 의미가 강하므로 보통 **법정화폐**로 두되, 필요하면 문맥 속에서 통화 팽창/통화 희석의 함의를 살린다.
- inflation은 문맥에 따라 **물가 인상**으로 단순화하지 말고, 저자가 통화팽창 의미로 쓰면 **통화 인플레이션** 또는 문맥 설명을 유지한다.
- deflation은 주류경제학적 부정 프레임으로 자동 번역하지 말고, 저자의 논지를 따라 **디플레이션의 긍정적 기능**이 드러나게 번역한다.
- sovereignty는 비트코인 맥락에서 **자기주권**이 더 적절할 때가 많다.
