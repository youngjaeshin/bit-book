# Summary template

## English chapter template

```md
# {Chapter Title}

## Chapter Digest

{2-5 paragraphs of readable, connected summary that preserves the chapter's argumentative flow and preferably states the author's claims directly rather than constantly saying "the author argues..."}

## Key Takeaways

- ...
- ...
- ...

## Notable Terms / Claims

- ...
- ...
```

## Korean chapter template

```md
# {챕터 제목}

## 읽는 느낌의 요약

{2-5문단. 저자의 논지를 직접 압축 서술하는 연결형 요약. "저자는 ~라고 말한다"의 반복을 피할 것}

## 핵심 포인트

- ...
- ...
- ...

## 중요 용어 / 주장

- ...
- ...
```

## Reminder

- 요약의 중심은 반드시 문단형 digest다.
- bullet은 보조다.
- 챕터마다 문제제기 → 전개 → 결론 흐름이 살아 있어야 한다.
- 독자 반응이나 감상 대신 **저자 주장과 논리 흐름**을 중심에 둔다.
- 가능하면 해설자 톤보다 저자 논지를 곧바로 압축한 문장으로 쓴다.
