# Windows/Mac/로컬 연동

## 요약

이 가이드는 전용 장비를 사지 않아도 데스크톱 기기 하나로 비트코인 노드 스택을 만들 수 있음을 보여 준다. 윈도우나 맥에 비트코인 코어나 노츠를 설치하고, 같은 기기에서 스패로우를 연결해 노드와 지갑이 로컬로 대화하게 만드는 흐름이다.

같은 기기에서 쓸 때는 스패로우가 127.0.0.1과 8332 포트를 그대로 쓰고, File > Settings > Server > Edit Existing Connection으로 들어가 연결한다. 이때 비트코인 데이터 폴더를 정확히 골라야 `.cookie` 파일에 있는 임시 RPC 사용자명과 비밀번호를 스패로우가 찾을 수 있다.

노드와 지갑이 같은 로컬 네트워크에 있을 때는 기기의 LAN IP와 bitcoin.conf 설정도 중요하다. 윈도우와 맥은 각각 방화벽 예외를 열어 줘야 하지만, 목표는 같다. 스패로우가 노드에 직접 닿게 하되, 연결은 제3자 서비스가 아니라 집 안의 네트워크 안에서 끝내는 것이다.

이렇게 구성하면 평범한 데스크톱이나 노트북도 자가 호스팅 백본이 된다. 사용자는 잔액을 검증하고 거래를 전파하며, 워치-온리 지갑 트래픽을 집 안 네트워크 안에 묶어 둘 수 있다.

이 가이드는 버튼 이름과 메뉴 경로를 외우는 데서 끝나지 않는다. 어떤 설정을 왜 바꾸는지, 어떤 입력값이 어떤 동작을 여는지, 어디서 검증해야 실수가 줄어드는지까지 운영 관점으로 묶어 준다. 그래서 독자는 단순히 절차를 따라 하는 수준을 넘어, 다음에 비슷한 상황이 와도 스스로 점검하고 복구하고 선택할 수 있는 기준을 얻게 된다.
