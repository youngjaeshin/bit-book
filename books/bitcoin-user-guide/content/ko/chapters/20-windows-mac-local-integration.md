# Windows/Mac/로컬 연동

## 요약

윈도우와 맥 사용자에게는 노드와 스패로우를 같은 기기나 같은 로컬 네트워크에서 붙여 쓰는 흐름이 제공된다. 이 장은 비트코인 코어 또는 노츠 설치, 필요할 때 RPC용 방화벽 개방, 그리고 스패로우가 로컬 노드와 깔끔하게 통신하도록 설정하는 과정을 다룬다.

두 운영체제에서 기본 구조는 같지만 필요한 단계는 조금씩 다르다. 윈도우에서는 제어판과 인바운드 규칙을 쓰고, 맥에서는 방화벽 상태를 확인한 뒤 필요하면 노드 앱을 허용하고, 그 다음 스패로우에서 로컬 IP와 bitcoin.conf에 적어 둔 RPC 자격 증명을 정확히 입력한다. 연결 테스트가 초록색으로 바뀌면 성공이다.

여기서 전해지는 메시지는 로컬 노드가 전용 장비만의 몫은 아니라는 점이다. 이미 가지고 있는 데스크톱이나 노트북도 로컬 연동을 제대로 해 두기만 하면, 감시와 지출을 떠받치는 자기보관의 백본이 될 수 있다.
