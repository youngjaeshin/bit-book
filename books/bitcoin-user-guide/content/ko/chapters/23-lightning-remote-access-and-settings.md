# 외부 접속 + 설정

## 요약

원격 라이트닝 제어는 수탁자에게 맡기지 않고 노드를 들고 다니는 방법이다. 이 장은 토르와 테일스케일을 써서 제우스를 로컬 네트워크 밖에서도 노드에 연결하게 하므로, 사용자는 기기 앞에 직접 서 있지 않아도 잔고를 확인하고 도구를 열고 결제를 관리할 수 있다.

뒤의 내용은 노드의 동작을 다듬는 데 집중한다. 네트워크 설정, 개인 설정, 채널 설정, 라우팅 규칙, 워치타워 옵션은 모두 노드가 실제로 쓰이기 시작했을 때 얼마나 안전하고, 얼마나 프라이빗하며, 얼마나 유용할지를 바꾼다. 그러므로 라이트닝 노드는 고정된 가전제품이 아니라 운영자의 습관에 맞게 조정해야 하는 시스템이다.

원격 접속과 설정을 함께 쓰면 라이트닝이 호스팅 상품이라기보다 개인 서비스처럼 느껴진다. 사용자는 키를 지키고, 라우팅 결정을 지키고, 상황이 바뀌면 방향도 바꿀 수 있다.
