# 19 Mempool Rpc And Reachable Node

## Chapter Digest

Mempool tools and RPC commands turn a node into something the user can interrogate directly. The chapter shows how to connect a mempool app, how to use an RPC explorer, and how to issue commands from the terminal so the node’s state can be inspected without guessing or waiting for a third-party website to explain it.

Reachability is the second half of the story. It works through the question of whether the node is reachable from outside, then shows how anonymous-network access, clearnet reachability, router port forwarding, ISP router settings, and firewall adjustments all affect whether inbound connections can actually happen. The point is not cosmetic exposure; it is genuine participation in the network.

By the end, the reader can check whether the node is useful only locally or whether it can be found and used by peers. That difference matters because it decides how much the user’s node contributes to the network rather than merely observing it.
