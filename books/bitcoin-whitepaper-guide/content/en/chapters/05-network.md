# Network

## Chapter Digest

The network section shows how Bitcoin turns many independently operating nodes into a single ledger without central direction. Nodes validate, relay, and compare information while the longest valid chain serves as the focal point of consensus. The power of the system lies not in each node knowing everything instantly, but in the network’s ability to converge on one history without a central authority.

The commentary uses this to explain why decentralization is not a slogan but an architectural property. A distributed network is hard to censor, hard to destroy, and able to continue despite missed messages, partitions, or temporary absence of participants. The network works because local validation and global convergence reinforce each other.
